# Pick a model

1. Open **Copilot Chat** (`Cmd+Shift+I` / `Ctrl+Shift+I`)
2. Click the **model picker** (top of the chat panel)
3. Select any model under the **CrofAI** group

### Reading the model list

Each model shows pricing at a glance:

- `↑$0.35/M ↓$1.70/M` — input / output cost per million tokens
- `$(gift) Free` — no cost models (Qwen3.5 9B, GLM 4.7 Flash)

### Reasoning effort

Models with reasoning support show **No Thinking / Low / Medium / High** buttons directly in the picker. Select the effort level before confirming the model.
