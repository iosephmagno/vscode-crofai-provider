# You're ready

The **CrofAI status bar item** shows your current credits and daily request quota. Click it anytime for a summary.

### Tips

- **Switch models** anytime — click the model name in the Copilot Chat header
- **Agent mode** works out of the box — file edits, terminal, search all supported
- **Vision** — drag images into Copilot Chat with Kimi, Gemma, or Qwen models
- **Refresh models** — run `CrofAI: Refresh Models` after CrofAI adds new models
